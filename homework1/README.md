# How to Run Homework 1
-----
### 1. Start the virtual environment (venv)
```bash
source venv/bin/activate
```

### 2. Move to correct directory
macOS/Linux  
```bash 
cd ~/cs4300/homework1
```
Windows  
```bash
cd %USERPROFILE%\cs4300\homework1
```
### 3. Run pytest
``` bash
python -m pytest
```

### Results of test will be shown in the terminal.
